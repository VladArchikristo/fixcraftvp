## NOTES FOR VIDEO MANAGER

### First 2–3 Lines Strategy
The first ~150 characters show before "Show more" — must hook immediately:

> "Stop losing money on the road. HaulWallet tracks every load, every mile, and every dollar — so you always know where your money went."

This line:
- States the benefit immediately (stop losing money)
- Names the app
- Explains what it does in one sentence
- No fluff, no "Welcome to our channel"

### SEO Keywords to Include
Primary keywords (in first 100 words):
- `truck driver app`
- `load tracking app`
- `mileage tracker for truckers`
- `trucker earnings tracker`

Secondary:
- `owner operator tools`
- `CDL driver app`
- `trucking expense tracker`
- `free trucking app`

### Timestamp Best Practices
- YouTube auto-creates chapter markers from timestamps if format is exact: `0:00 — Title`
- Keep timestamp titles short (<40 chars)
- First timestamp must be `0:00`

### Hashtag Strategy
- Use 3–5 hashtags max in the top section (YouTube shows these above description)
- Full hashtag list goes at the bottom
- Top hashtags recommended:
  ```
  #TruckDriver #Trucking #HaulWallet
  ```

If targeting Spanish-speaking market (US Latino truckers), create a second description version in Spanish. High-volume keywords: `app para camioneros`, `rastreador de cargas`, `ganancias camionero`.
