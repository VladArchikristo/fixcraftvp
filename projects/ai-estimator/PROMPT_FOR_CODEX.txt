Build a complete Telegram bot for a handyman business called FixCraft VP in Charlotte, NC. The bot turns job-site photos into detailed cost estimates using AI.

WORKING DIRECTORY: Create a new folder ~/fixcraft-estimator-bot/ and build everything there.

### 4. database.py
SQLite database with tables:
- estimates (id, chat_id, photo_path, description, ai_analysis JSON, line_items JSON, total_amount, status, created_at)
- pricing_config (id, labor_rate_standard, labor_rate_emergency, material_markup, overhead_percent, profit_margin, minimum_service_call, updated_at)

Include functions: save_estimate(), get_estimates_by_chat(), get_all_estimates(), update_pricing_config(), get_pricing_config()

### 5. ai_analyzer.py
Function analyze_photo(photo_path: str, description: str) -> dict:
- Read image file as base64
- Call OpenAI GPT-4o with vision capabilities
- System prompt: "You are an expert handyman estimator with 20+ years experience in Charlotte, NC. Analyze the photo and return STRICT JSON with: job_type, title, description, materials list (item, qty, unit, unit_cost, notes), labor_breakdown (task, hours), total_labor_hours, difficulty, hidden_issues, permit_required, notes."
- Parse JSON response and return dict
- Handle errors gracefully

### 6. pricing.py
Function calculate_estimate(ai_analysis: dict, pricing_config: dict, is_emergency: bool = False) -> dict:
- Calculate materials subtotal and with markup
- Calculate labor subtotal (hours * rate)
- Add overhead (15% on subtotal)
- Add profit margin (20% on subtotal)
- Apply minimum service call if total < minimum
- Return complete breakdown dict

### 7. pdf_generator.py
Function generate_pdf(estimate_data: dict, output_path: str):
- Use ReportLab
- Create professional branded PDF with FixCraft VP header
- Include job description, materials table, labor table, overhead, profit, total
- Add terms and disclaimer
- Add footer with contact info
- Save to output_path

### 8. bot.py (MAIN FILE)
Build async Telegram bot with python-telegram-bot v20:

Commands:
- /start - Welcome message explaining how to use
- /help - Instructions
- /history - Show last 10 estimates for this chat
- /stats - Show total estimates and revenue (admin only)
- /settings - Show current pricing config (admin only)
- /setprice - Update pricing values (admin only)

Photo handler:
- When photo received, download it
- Send "🔍 Analyzing photo..." message
- Call ai_analyzer.analyze_photo()
- Call pricing.calculate_estimate()
- Format and send estimate message
- Call pdf_generator.generate_pdf()
- Send PDF as document
- Save to database
- Send confirmation

Admin checks: Compare chat_id to ADMIN_CHAT_ID from env.

Error handling: Wrap everything in try/except. If AI fails, send friendly error message.

Rate limiting: Track requests per chat_id, max 10 per hour.

Photo cleanup: Delete downloaded photos after 24 hours.
