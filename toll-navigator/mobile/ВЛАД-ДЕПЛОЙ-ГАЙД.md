# 🚀 ДЕПЛОЙ В GOOGLE PLAY (шаг за шагом)

Когда Клод закончил и ты проверил — осталось загрузить в Google Play.

## Шаг 1: Найди AAB файл

Клод должен был сказать где файл. Обычно это:
```bash
ls -lh /tmp/haulwallet-*.aab
```
Если файла нет — скачай с Expo:
1. Открой ссылку которую дал Клод (build URL)
2. Скачай `.aab` в папку Загрузки

## Шаг 2: Запусти скрипт загрузки

```bash
cd "/Users/vladimirprihodko/Папка тест/fixcraftvp/toll-navigator/mobile"
python3 upload_aab_only.py
```

Скрипт спросит путь к файлу — укажи его.

## Шаг 3: Активируй в консоли (только для первого релиза)

Если приложение ещё в статусе "Draft":
1. Открой https://play.google.com/console
2. Зайди в HaulWallet → Test and release → Internal testing
3. Найди новый релиз → нажми "Review release"
4. Нажми синюю кнопку "Save and publish"

## Шаг 4: Проверь на телефоне

Через 5-15 минут:
- Открой Play Store → HaulWallet
- Должна появиться кнопка "Update"

Если не появилась:
```
Настройки → Приложения → Google Play Store → Хранилище → Очистить кэш
```

## ⚠️ Если скрипт не сработал

Отправь Hermes'у:
1. Точный путь к AAB файлу
2. Вывод ошибки из скрипта
3. Он загрузит сам
