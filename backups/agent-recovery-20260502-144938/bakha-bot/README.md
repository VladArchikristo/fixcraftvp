# Бахтияр / Баха Bot

Чистый Telegram bot, созданный с нуля. Не копия других ботов.

## Роль
Бахтияр (Баха) — сильный IT-инженер, senior/staff-level programmer-architect.

## Запуск
1. Создать нового бота в @BotFather.
2. Записать токен в `.env`:

```env
BAKHA_BOT_TOKEN=123456:ABC
ALLOWED_USER_ID=244710532
```

3. Проверить GPT-5.5 proxy на `http://127.0.0.1:10531/v1`.
4. Запустить `python3 bot.py` или LaunchAgent `com.vladimir.bakha-bot`.
