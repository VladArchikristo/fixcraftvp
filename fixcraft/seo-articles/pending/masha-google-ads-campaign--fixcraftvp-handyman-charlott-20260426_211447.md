## Сайт как лендинг для Ads — ✅ Технически готов

- Быстро грузится (Vercel CDN)
- Есть телефон, адрес, услуги

**Что слабо прямо сейчас:**
- Нет социального доказательства (отзывы = 0)
- Нет чёткого CTA "above the fold" на мобиле
- Нет страницы с ценами (люди уходят без цены)

---

## Трафик из Ads → органический рост?

**Прямой связи нет.** Google не повышает позиции из-за платного трафика.

**Но косвенно — да:**

| Эффект | Как работает |
|--------|-------------|
| Поведенческие сигналы | Люди кликают → проводят время на сайте → Google видит "сайт полезен" |
| Брендовые поиски | После рекламы люди ищут "FixCraftVP" → это органика |
| Отзывы | Клиенты из Ads → оставляют GBP отзывы → рост в картах |
| Backlinks | Клиенты пишут о тебе в Nextdoor/Facebook → ссылки |
