# FixCraft VP — Multi-Agent Architecture (Hermes Army)
## Оркестратор + Специалисты на базе Hermes

### Концепция
Вместо одного универсального Claude Code — армия узкоспециализированных агентов Hermes.
Каждый агент живёт в своём профиле, имеет свою память, свои навыки и может работать автономно.
Оркестратор (ты, главный Гермес) — центральный мозг: получает задачи от Влада, 
распределяет по специалистам, собирает результаты.

## 🔄 Как работает оркестрация

### Способ 1: Файловый обмен (надёжно, просто)
1. Влад: "Нужна статья про TV mounting"
2. Оркестратор создаёт `tasks/masha_001.json` с заказом
3. Маша (cron каждые 15 мин) проверяет tasks/, берёт заказ, пишет статью
4. Маша сохраняет в `content/tv-mounting-2026-04-26.md`
5. Маша отмечает task как done
6. Оркестратор видит done и говорит Владу: "Готово!"

### Способ 2: Delegate Task (быстро, интерактивно)
1. Влад: "Нужна статья"
2. Оркестратор вызывает `delegate_task(goal="Напиши статью...", skills=["seo-writing"])`
3. Под-агент выполняет и возвращает результат
4. Оркестратор показывает Владу

### Способ 3: Cron-агенты (полная автономия)
- Маша: каждый день в 9:00 генерирует 1 статью по календарю
- SEO-агент: каждый понедельник рассылает outreach
- Email-агент: каждые 30 мин чекает почту
- Все работают без приказа, архитектор только контролирует
